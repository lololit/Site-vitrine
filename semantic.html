Ici nous allons "Tokeniser" notre flux de data en isolant les différents caractères (string, emoticones, mentions, Tags HTML etc...)

import re
 
emoticons_str = r"""
    (?:
        [:=;] # Eyes
        [oO\-]? # Nose (optional)
        [D\)\]\(\]/\\OpP] # Mouth
    )"""
 
regex_str = [
    emoticons_str,
    r'<[^>]+>', # HTML tags
    r'(?:@[\w_]+)', # @-mentions
    r"(?:\#+[\w_]+[\w\'_\-]*[\w_]+)", # hash-tags
    r'http[s]?://(?:[a-z]|[0-9]|[$-_@.&amp;+]|[!*\(\),]|(?:%[0-9a-f][0-9a-f]))+', # URLs
 
    r'(?:(?:\d+,?)+(?:\.?\d+)?)', # numbers
    r"(?:[a-z][a-z'\-_]+[a-z])", # words with - and '
    r'(?:[\w_]+)', # other words
    r'(?:\S)' # anything else
]

tokens_re = re.compile(r'('+'|'.join(regex_str)+')', re.VERBOSE | re.IGNORECASE)
emoticon_re = re.compile(r'^'+emoticons_str+'$', re.VERBOSE | re.IGNORECASE)

def tokenize(s):
    return tokens_re.findall(s)

def preprocess(s, lowercase=False):
    tokens = tokenize(s)
    if lowercase:
        tokens = [token if emoticon_re.search(token) else token.lower() for token in tokens]
    return tokens

Analyse de sentiments¶

On importe la librairie TextBlob (installer via les commandes pip ou conda si vous n'arrivez pas à l'importer). Celle-ci va nous permettre de faire une analyse de sentiment avec la notion de polarité et de subjectivité qui la compose.

In [17]:
from textblob import TextBlob

blob = TextBlob(tweet_text_as_string)

tweets_sentiment = blob.sentiment
tweets_polarity = tweets_sentiment[0]
tweets_subjectivity = tweets_sentiment[1]
In [18]:
print(" Polarity :" , tweets_polarity)

L'analyse de la polarité des tweets démontre une certaine neutralité de ceux-ci la valeur 0 étant neutre, -1 négatif, 1 positif. Sur un sujet sensible comme le moyen-orient cela peux nous montré une certaines opposition entre les points de vues, les soutiens à différents partis et pays.

In [19]:
print(" Subjectivité :" , tweets_subjectivity)

 Subjectivité : 0.3887217370751851

Etant donné que la data provient de twitter, nous partons d'un constat de base qui est qu'il peut y avoir beaucoup de tweet subjectif comme des simples liens. Pourtant ici notre analyse de subjectivité nous démontre que nos tweets sont plutôt objectifs car plus proche de la valeur 0 que 1. Nous pouvons donc dire qu'il y a de vrais échanges concernant ce sujet sur twitter, malgré une minorité de tweets subjectifs.
